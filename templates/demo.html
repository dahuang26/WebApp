Our Work

This website application provides an auto design tool for mass timber rocking wall system, this procedure will optimize design utilizing genetic algorith (GA). A generalized ANN model will be used for max response prediction. Users only need to input a few parameters of the building and design targets they want to design, and the application will automatically generate optimized design result.

1. Input building site information. 2. Input basic building parameters. 3. Input PBSD design target. 4. Run GA and ANN model. 5. Generate optimized results.

Design procedure

Site information; Building story number; Story weight; Story height; Elastic modulus of wall panel;Damping ratio;
